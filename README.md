# spring-boot-challenge
spring-boot 와 spring-data-jpa를 이용한 쇼핑몰 구현하기

+ 프로그램 기능 목록
  + 로그인
  + 회원가입
  + 카테고리 별 상품조회
  + 주문 기능(시뮬레이션)
  + 게시판
  + 공지사항

+ 구현해야 할 것들
  + 상품 정보 수집 (쇼핑몰 크롤링) V
  + 뷰 작성 
  + js 파일 작성
  + 상품 조회 API 작성
  + 페이징 기능 구현
  + 로그인 및 회원가입 기능
  + 게시판 글 조회 API 작성
  + 공지사항 글 조회 API 작성
  + 필요한 모든 로직에 대한 단위테스트 작성
  
+ 개발 환경
  + 개발 언어 : Java
  + 프레임워크 : Spring-boot, Spring-data-JPA
  + 서버 : google cloud platform
  + 데이터베이스 : gcp sql (MySQL)
  + 개발 일시 : 2020/06/21 ~
